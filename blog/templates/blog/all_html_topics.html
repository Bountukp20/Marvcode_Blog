As a beginner to programming, the first language you would probably come across is HTML, which is the most basic programming language. The Hyper text Markup Language, which is the meaning to the acronym HTML. It is a standard markup language for creating web pages. HTML, like a human body, consist of elements and atttributes. A human body consist of a the head, body and foot e.t.c. Just as in a human body the head can't come after the body e.t.c, so in HTML there is a syntax of writing codes. HTML Document 


                            <!doctype html>
                                <html>
                                    <head>
                                    <title>Page Title</title>
                                    </head>
                                    <body>
                                
                                        <h1>My First Heading</h1>
                                        <h2>My First Heading</h1>
                                        <h3>My First Heading</h1>
                                        <p>My first paragraph.</p>
                                
                                    </body>
                                </html>

The <!doctype html> is a declaration message that defines the type of HTML document it is i.e html 5, html 4, e.t.c. The <html> element is the root element of an HTML page. The <head> element contains meta information about the web page. The <title> element specifies a title for the HTML page (which is usually shown in the browser's title bar or in the page's tab). The title can also be used as an attribute which I will be coming to soon. The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. The <h1> element defines a large heading. There are h1 to h6, and each of them makes the text look smaller in the descending order i.e that is h1's text is bigger than h2's, h2's bigger than h3's e.t.c. The <p> element defines a paragraph. There several of them e.g <div></div>: which divides contents, and puts space around the wrapped contents. <span></span>: which is almost like the div tag, but it takes the amount of spaces needed. <br>: It jumps two lines between two contents. <hr>: It draws an horizontal line across a web page. <a></a>: Or anchor tag, it is used to create a hyper link between documents, websites e.t.c. This is different from the link tag. <link>: Defines the relationship between the current document and an external resource. It is an empty element, it contains attributes only. <script></script>: It is used to embed javascript code in HTML document. <table></table>: It is used to define a table content in a HTML document. <tr>: It defines rows in an HTML document. <input>: It is used to create interactive controls for web-based forms in order to accept data from the user; a wide variety of types of input data and control widgets are available, depending on the device and user agent. <p><strong></strong></p>: it means that the tag is nested properly. <p><strong></p></strong>: It means that the tag is not nested properly. <b></b>: It is used to make text contents bold. <em></em>: It is used to emphasize text contents. <mark></mark>: It is used to highlight text contents. <abbr></abbr>: It is used to display texts in an abbreviated form. <img></img>: It is used to insert images to web page. <label></label>: It is used to name an input field. <button></button>: It is used define a clickable button. <form></form>: It is used to define a form. <footer></footer>: It is used to define a footer for a document or section. : It is used to define a comment. <bdo>: It is used to override the current text direction. <audio></audio>: it used to define embedded sound content. <nav></nav>: It is used to define navigation links. <section></section>: It is used to define a section in a document. <header></header>: It is used to define a header for a document or section <option></option>: It is used to define an option in a drop-down list. <select></select>: It is used to define a drop-down list.

What is an HTML Element?
The HTML Element can simply be referred to as the human part i.e head, body, and foot e.t.c which contains components needed for life, in this case needed to display content i.e: text e.t.c. An element is catagorised into two: open tags, and closed tags. Closed tags: They are HTML elements that has a start tag, some content in it, and a closed tag i.e: <h1>My First Heading</h1>. Open tags: They are HTML elements that just has a start tag. There are no contents written in them, as most times there are other elements that follows on a new line. They can be referred to as empty tags which only contains attributes most often e.g <br>, <hr> e.t.c. What are attributes? They are written inside a HTML element i.e 


                        <body style="background: grey;">
                         <a href="https://www.google.com"  title="google">
                                Google
                         </a>
                        </body>

The value inside the tags are called attributes. As shown an element can contain one or more attributes. Attributes gives more information to an element about a content is to be displayed, just like a skin colour of someone tells us who is dark or who is light.
The importance of HTML
HTML is the fundamental language for creating web related things e.g web pages, web games, web apps e.t.c. As shown in the examples you must have noticed spaces in between open tags when writting a child elements e.g 


                            <body>
                
                                <h1>My First Heading</h1>
                                <p>My first paragraph.</p>
                
                            </body>

This process is called indenting. It is very useful as it makes your code easier to read. Also HTML is not case sensitive, so it is a conventional way of written codes in HTML.

Default Homepage
The index.html page is the most common name used for the default page shown on a website if no other page is specified when a visitor requests the site. In other words, index.html is the name used for the homepage of the website.

Site Architecture and Index.html.
Websites are built inside of directories on a web server. For your website, you must save each webpage as a separate file. For example, your "About Us" page may be saved as about.html and your "Contact Us" page may be contact.html. Your site will be comprised of the .html documents. Sometimes when someone visits the website, they do so without specifying one of these specific files in the address that they use for the URL. For example: http://www.lifewire.com Even though there is no page listed in the URL request made to the server, that web server still needs to deliver a page for this request so that the browser has something to display. The file that will be delivered is the default page for the directory. Basically, if no file is requested, the server knows which one to serve up by default page in a directory is named; index.html In essence, when you go to a URL and specify a specific file, that is what the server will deliver. If you do not specify a file name, the server looks for a default file and displays that automatically -almost as if you had typed in that file name in the URL.

Other Default Page Names.
Besides index.html, there are other default page names that some sites use, including: .index.html .defualt.html .home.html or .htm
